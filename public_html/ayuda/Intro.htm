Introducción

Bienvenido al manual de usuario del sistema de investigación laboral de RH Global.

Este sistema te permite iniciar y monitorear el proceso de investigación laboral de tus candidatos desde una interfaz fácil de usar, basada en internet y con manejo de tu información de forma sencilla y amigable.

El resultado final de cada proceso de investigación es un dictamen emitido en forma de reporte, descargable en formato PDF para su fácil impresión y archivo.

Requerimientos

El sistema, basado en web, requiere una conexión con internet para su uso, además de un navegador de última generación, ya que utiliza tecnología de punta en el procesamiento de los datos capturados, lo que permite un flujo de trabajo (captura) altamente dinámico y sin necesidad de recargar la página cada vez que se presiona un botón ó enlace.

Los requerimientos mínimos recomendados son los siguientes:

PC Pentium 3 ó superior
256 Mb de Ram o superior
Microsoft Internet Explorer 5.5 ó superior

El sistema ha sido probado exitosamente con otras configuraciones, en sistema operativo Mac OSX usando el navegador Firefox, también Firefox para windows puede ser usado, Netscape Navigator 7.0 y Mozilla son otros navegadores que soportan nuestra aplicación.
