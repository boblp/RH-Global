Procesar Candidato

Para comenzar el proceso de investigación, es necesario que indiques cuando un candidato está listo para comenzar el proceso. Para hacerlo da click en el icono del candidato correspondiente.

 

Una vez hecho esto, aparecerá el siguiente diálogo:

 

Al presionar [Aceptar], el candidato será enviado a nuestros operadores para comenzar con el proceso. A partir de este momento, podrás monitorearlo desde la página "En Proceso" ó "Terminados" según sea el caso.
