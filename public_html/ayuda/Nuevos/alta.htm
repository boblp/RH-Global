Alta de Candidatos

Para agregar un candidato, debes presionar primero el botón "Agregar Candidato" que aparece en la parte inferior del listado, una vez que lo haces, aparecerá en el área de edición (derecha) el siguiente formulario:

Pantalla de captura de candidato

 

Este formulario cuenta con los campos necesarios para que el personal de investigaciones pueda realizar correctamente su trabajo.

En la eventualidad de que la información para algún campo se desconozca momentaneamente, puede guardarse el candidato para continuar editándolo más tarde.

Los datos imprescindibles son el nombre y la fecha de nacimiento del candidato, sin embargo TODOS los campos son necesarios para asegurar que el proceso de investigación resulte en información confiable.

En la imágen de la derecha se muestran los campos tal como deben ser llenados.

Se debe tener especial cuidado en capturar el nombre completo (nombres y apellidos completos, sin abreviaciones), la fecha de nacimiento (cuatro dígitos para el año, y dos dígitos para el mes y el día) donde el orden es AÑO-MES-DIA, y el número de seguro social (11 dígitos).

 

Editar Candidato

Este mismo formulario es presentado cuando se edita un candidato. Para abrir el formulario de edición, se debe presionar el ícono en el renglón al candidato correspondiente.

Las mismas reglas de edición aplican para la alta y la edición de un candidato.

 

 

Es importante resaltar que aunque el sistema puede permitir el procesamiento del formulario aunque no todos los campos se hayan llenado, se recomienda llenar todos los campos para garantizar su correcto procesamiento e investigación.
